About The Project:

Crypto Info is a responsive web application to provide real-time insight into the ever-evolving world of cryptocurrencies. The site allows users to effortlessly search and track various coins, with prices updated dynamically through live API calls. The interactive chart and intuitive interface make it easy to stay informed, while features like the favorites list offer personalized user experiences.

About Me:
